# Design System

## Tasarım Hedefi
Profesyonel teknoloji şirketi hissi.
Gereksiz kalabalık yok.
Sessiz güç, temiz hiyerarşi, net kontrast.

## Stil Özeti
- minimal
- modern
- kurumsal
- teknoloji odaklı
- premium ama gösterişsiz

## Renk Sistemi
### Ana Arka Plan
- neredeyse siyah veya çok koyu lacivert
- düz siyah yerine hafif mavi alt tonlu koyu yüzey tercih edilebilir

### Metin
- ana metin: beyaz / kırık beyaz
- ikincil metin: soğuk gri

### Vurgu Rengi
Tek vurgu rengi kullanılmalı.
Birden fazla güçlü vurgu rengi kullanılmamalı.

Öneri:
- ana vurgu: turkuaz / cyan-tinted teal

Altın ancak çok küçük detaylarda kullanılabilir.
Ana vurgu ile yarışmamalı.

## Typography
### Başlık Fontu
Montserrat

### Metin Fontu
Inter

## Tipografi Hiyerarşisi
- H1: güçlü, net, geniş boşluklu
- H2: bölüm başlıkları için belirgin
- H3: kart / alt başlıklar
- body: okunabilir, rahat satır yüksekliği
- small/meta: durum etiketi, küçük notlar

## Görsel Dil
- yumuşak glow
- ince grid
- hafif çizgisel network hissi
- düşük yoğunluklu hareket
- ağır 3D veya ucuz parlayan efektler yok

## Motion Kuralları
- kısa süreli hover
- kartlarda hafif translate / border glow
- section reveal motion hafif olmalı
- scroll yorgunluğu yaratacak uzun animasyonlar olmamalı

## Border & Radius
- kartlarda modern radius
- hafif border
- bazı kartlarda glass hissi olabilir ama aşırı blur kullanılmamalı

## Layout Kuralları
- geniş ama kontrollü container
- section spacing tutarlı
- fazla uzun boşluk yok
- mobilde bozulmayan stack düzeni

## UI Hissiyatı
Hedef referans hissi:
- teknoloji şirketi
- dijital altyapı şirketi
- modern SaaS / innovation studio
- güven veren startup ama oyuncak değil
