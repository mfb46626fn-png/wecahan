# WeCaHan Corporate Site Docs

Bu klasör, `www.wecahan.com` kurumsal ana marka sitesinin yeniden yapılandırılması için hazırlanmış uygulama dokümantasyonudur.

## Amaç
Bu doküman seti, Cursor içinde parça parça ve kontrollü şekilde ilerlemek için hazırlanmıştır.

Bu site:
- ana marka / kurumsal vitrin sitesi olacak,
- hizmet ve yetkinlik odaklı anlatı taşıyacak,
- seçili projeleri kontrollü biçimde gösterecek,
- dağınık girişim listesi gibi görünmeyecek.

## Aktif Proje Kapsamı
Şimdilik ana sitede sadece şu 3 proje yer alacak:
1. AAA Automation — `Active`
2. Prificient — `In Development`
3. Prificient Tools — `Active`
